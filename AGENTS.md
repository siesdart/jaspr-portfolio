## Melos: PRIORITIZE Melos scripts

The project is structured as a monorepo using [Melos](https://melos.invertase.dev/) to manage packages and dependencies. Melos is a CLI tool used to help manage Dart projects with multiple packages.
Before executing any CLI commands, find the relevant Melos script in `pubspec.yaml`(e.g., `melos serve`, `melos build`, etc), and if one exists, use it instead.

### Melos commands

- **Install dependencies:**
  `melos bootstrap`

- **Run the development server:**
  `melos serve`

- **Build the application for production:**
  `melos build`
