사내 백오피스 웹 서비스의 프론트엔드 개발 및 유지보수

1. 게임 개발 팀의 효율적인 콘텐츠 제작 및 모니터링 등 종합적인 게임 개발/운영을 돕기 위한 Next.js 기반 게임 콘솔 개발
2. KPI 등 대용량 게임 데이터를 시각화하여 화면에 안정적으로 렌더링하고, 사내 의사결정을 지원하는 Next.js 기반 데이터 플랫폼 개발
   - Tableau Embedding API를 사용한 Tableau 뷰 컴포넌트 제작
   - 각 게임 데이터 뷰가 별도의 페이지로 하드코딩된 것이 장기적인 관점에서의 개발과 운영에 어려움이 될 거라 판단해 이를 동적 세그먼트 페이지 하나로 정리하고 JSON 형식의 설정 파일을 만들어 해당 설정에 기반해 각 데이터 페이지가 동적으로 렌더되도록 하는 `Configuration Driven Development(CDD)` 방식으로 전체적인 프로젝트 구조 재설계
   - 이외에도 코호트 필터 등 잦은 수정이나 추가가 필요한 설정을 하드코딩이 아닌 통합된 JSON 파일에서 가져오도록 수정하여 대부분의 케이스에서 코드 수정 없이 JSON 만으로 작업이 가능하도록 워크플로우 정리
   - Next.js의 `generateStaticParams` 함수를 활용해 빌드 과정에서 각 데이터 뷰 페이지가 SSG로 생성되도록 하여 페이지 로딩 속도 개선
3. UI 일관성 유지 및 중복 개발 리소스 최소화를 위한 Tailwind CS 기반 20여 종의 컴포넌트로 이루어진 사내 공통 React UI 패키지 개발
   - 컴포넌트 개발 과정에서 코드 변경에 따른 실시간적인 컴포넌트 변화를 시각적으로 보고 테스트하는데 어려움을 느껴 DX 향상을 위해 Storybook 리서치 후 도입
   - tsup을 사용하여 번들링 후 사내 npm 레지스트리에 배포되도록 CI/CD를 구축하였으나, 번들링 시 Next.js 프로젝트에서 `use client` 등 지시자가 인식되지 않는 문제가 있어 `bundle` 옵션을 꺼 번들링 없이 각 파일의 트랜스파일만 진행하도록 수정
   - 사내 구성원들이 패키지 내 컴포넌트의 정보를 알고 쉽게 사용할 수 있도록 패키지 배포 시 Storybook 역시 정적 사이트로 빌드되어 사내에 배포되도록 CI/CD 작성
4. 다수의 독립적인 백오피스 서비스 간 일관된 사용자 경험 제공 및 서비스 간 통합 관리를 위한 공통 GNB(Global Navigation Bar) 패키지 개발
5. UI, GNB 등 사내 패키지의 에이전트 통합 개발 생산성 향상을 위한 Documentation 작성 및 MCP 웹 허브 구축
   - 패키지 별 작성된 Documentation에 대한 정보를 시각적으로 보고 버튼 원클릭으로 해당 Documentation을 읽을 수 있는 MCP의 설치 명령어를 복사할 수 있도록 웹 허브 구현
   - 패키지 업데이트 시 패키지 내 변경 사항 탐지 -> Documentation 수정을 자동화하는 커스텀 커맨드 작성
   - UI 패키지의 경우 Documentation을 배포된 Storybook에서도 볼 수 있도록 Storybook 내 컴포넌트 제작 후 연동
6. 제작한 웹 기반 게임의 배포 환경을 사내 누구나 손쉽게 테스트하고 결과물을 확인할 수 있도록 돕는 Next.js 기반 정적 웹 사이트 배포 플랫폼 개발
   - `@aws-sdk/client-s3` 패키지를 사용하여 사용자가 웹 사이트의 zip 파일을 업로드하면 압축을 풀어 aws s3 버킷에 업로드하고 버킷 내 웹 사이트 목록 및 접속 URL을 화면에 렌더하도록 구현
7. UA 팀의 지속적이고 빠른 광고 영상 제작을 돕기 위한 Next.js, Veo 3 기반 AI 영상 생성 플랫폼 개발
   - 사용자가 마우스/키보드 클릭만으로 동영상을 여러 세그먼트 구간으로 분리할 수 있는 타임라인 컴포넌트 제작
   - Veo 3 API에 전달할 프롬프트를 작성/수정하는 텍스트 에디터 제작
8. 사용자의 자연어 요구사항을 기반으로 사내 UI 및 GNB 패키지 등을 사용해 백오피스 프로토타입을 신속하게 빌드하는 에이전트 기반 하네스 구축
   - UI, GNB 등 사내 패키지들의 Documentation에 접근하는 방법을 담은 에이전트 스킬 작성
   - 에이전트를 디자이너, 개발자, 리뷰어 등의 역할로 나누어 사용자의 요구사항을 Next.js와 사내 패키지들을 사용하여 구현하도록 커맨드 작성
